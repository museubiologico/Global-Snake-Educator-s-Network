# Global-Snake-Educator-s-Network
The Global Snake Educators Network repository aims to share resources that strengthens education, supports conservation, and raises global awareness about snakebites to improve snake understanding and reduce human-wildlife conflict. 

This is a joint initiative of Butantan Biological Museum and Save the Snakes.

If you want to collaborate with our network or access more information, please connect to the project website:
https://sites.google.com/savethesnakes.org/global-snake-educators-network?usp=sharing
